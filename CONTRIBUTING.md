# Contributing

Contributions that improve multilingual quality, retrieval evaluation, safety, observability, or documentation are welcome.

1. Create a branch from `main`.
2. Install development dependencies with `python -m pip install -r requirements-dev.txt`.
3. Make a focused change and add tests.
4. Run `ruff check .` and `pytest`.
5. Open a pull request describing the problem, approach, tests, and safety impact.

Do not submit real customer data, private company material, credentials, production endpoints, or copyrighted datasets without permission. Use synthetic or appropriately licensed examples.
